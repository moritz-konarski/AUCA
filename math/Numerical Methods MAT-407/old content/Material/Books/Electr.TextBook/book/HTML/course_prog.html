SILLABUS для программы "SOFTWARE ENGINEERING"
II, III курсы, II семестр (16 января - 10 мая 2006 г.)

  1. Преподаватель: Скляр Сергей Николаевич, проф., д.ф.-м.н.

  2. Объем занятий: 2 пары в неделю (1 пара-лекция + 1 пара-лабораторная практика), общее количество - 64 часа, 3 кред., 16 раб. недель.

  3. Консультации: согласно индивидуальному графику работы преподавателя. Офис: I/332, тел. 66-15-47.

  4. Цель курса. Основная цель - дать студентам знания и навыки использования вычислительных методов при решении разного рода прикладных задач, таких как: численное интерполирование и интегрирование, решение систем алгебраических уравнений, численное решение задач для дифференциальных уравнений.

  5. Краткое описание курса. Изучаются вычислительные методы и алгоритмы, используемые для решения задач, возникающих при моделировании физических процессов, а также в экономических приложениях математики. Курс включает два основных раздела вычислительной математики: численные методы алгебры и анализа и методы решения задач для дифференциальных уравнений. Студенты познакомятся со следующими темами: особенности компьютерной арифметики; интерполяция и аппроксимация функций многочленами и сплайнами; прямые и итерационные методы решения систем линейных и нелинейных алгебраических уравнений; квадратурные формулы для вычисления интегралов; разностные схемы для решения обыкновенных дифференциальных уравнений, а также методы исследования этих разностных схем. Помимо лекционных, предполагаются лабораторные занятия в компьютерном классе.

    Для изучения данного курса необходим определенный уровень знаний по следующим дисциплинам: линейная алгебра, векторный анализ и аналитическая геометрия, математический анализ, элементы теории обыкновенных дифференциальных уравнений. Кроме того, нужны некоторые навыки программирования на одном из языков высокого уровня.

  6. Prerequisites: MAT 315 и COM 111.

  8. Программа курса.

    А) Содержание лекционного курса.
    Введение в предмет. Математическое моделирование и вычислительная математика.
    Численные методы алгебры и анализа.

  1. Основные понятия и определения вычислительной математики (I).

    • Пространства векторов и сеточных функций, норма и ее свойства, эквивалентные нормы.

    • Норма матрицы, свойства нормы матрицы.

    • Число обусловленности матрицы и его свойства. Теорема об относительной погрешности возмущения матрицы.

    • Классификация вычислительных погрешностей. Погрешности округления вещественных чисел и их трансформация под воздействием арифметических операций. Погрешность дискретизации.

  2. Интерполяция сеточных функций.

    • Интерполирование сеточных функций алгебраическими многочленами. Многочлен Лагранжа. Оценка погрешности интерполяции многочленом Лагранжа. Сходимость интерполяционного процесса. Общая постановка проблемы оптимизации методов решения задач из некоторого класса. Оптимальный выбор узлов интерполяции. Многочлены Чебышёва.

    • Интерполирование сеточных функций при помощи сплайна. Определение сплайна. Построение линейного и кубического сплайнов, оценки погрешности интерполяции.

  3. Численное интегрирование.

    • Квадратурные формулы составного типа. Формулы прямоугольников, трапеций, Симпсона. Оценки погрешностей.

  4. Прямые методы решения системы линейных алгебраических уравнений.

    • Метод исключения Гаусса. Метод прогонки: прогонка стандартная и прогонка Дорра. Теорема об устойчивости и корректности стандартной прогонки.

    • Влияние погрешностей округления на реализацию метода прогонки.

  5. Решение нелинейных алгебраических уравнений и систем итерационными методами.

    • Сжимающие функции. Теорема о сжимающей функции. Метод простой итерации.

    • Метод Ньютона и его геометрическая интерпретация. Модифицированные варианты метода Ньютона, метод секущих, их геометрическая интерпретация. Оценка сходимости метода Ньютона.

    • Метод Хэлли и его модификация.


Методы численного решения дифференциальных уравнений
Основные понятия и определения вычислительной математики (II).

  • Сеточные операторы: операторы разностного дифференцирования и их свойства.

  • Сеточные операторы монотонного вида: теорема сравнения, оценка решения сеточного уравнения с оператором монотонного вида.

  • Аппроксимация дифференциального уравнения разностным. Устойчивость и сходимость разностной задачи, теорема Лакса о сходимости.

  1. Методы численного решения задачи Коши для нелинейного дифференциального уравнения первого порядка. Методы Рунге – Кутта первого, второго и третьего порядков точности, исследование сходимости.

  2. Построение и анализ разностных схем для линейных дифференциальных уравнений первого и второго порядков.

  • Метод конечных разностей. Построение разностных схем для уравнения первого порядка. Первая краевая задача для уравнения диффузионно-конвективного переноса: схемы с центральной и направленной разностями, схемы А.А.Самарского и А.М.Ильина. Монотонные разностные схемы. Исследование аппроксимации и устойчивости, оценки сходимости.


Б) Содержание лабораторной практики.

Лабораторная практика проходит в компьютерной лаборатории и тематически сориентирована в следующих двух направлениях.

  1. Работа с пакетом обучающих программ «ЧИСЛЕННЫЕ МЕТОДЫ».

    Пакет «ЧИСЛЕННЫЕ МЕТОДЫ» включает текстовые файлы с необходимой теорией (краткое описание используемых методов и их свойств) и заданием, которое требуется выполнить, работая по данной теме, а также набор тестовых задач. Вычислительные эксперименты с Пакетом позволяют проиллюстрировать основные разделы лекционного курса и выявить особенности, которыми обладают изучаемые численные методы и алгоритмы при их использовании в «реальных» условиях. Используя Пакет «ЧИСЛЕННЫЕ МЕТОДЫ» студент не занимается самостоятельным программированием, что позволяет существенно сэкономить время, отпущенное для изучения предмета. Письменный Отчет должен быть представлен преподавателю после завершения работы по каждой теме.

    Разделы курса, которые включены в пакет обучающих программ «ЧИСЛЕННЫЕ МЕТОДЫ»:

  1. Интерполяция функций. Работая с этой программой, студент может сравнить различные виды сплайн-интерполяции (кубический и кусочно-линейный сплайны), используя при этом как равномерную, так и неравномерную адаптированную сетку.

  2. Численное интегрирование. Программа позволяет проиллюстрировать работу различных квадратурных формул составного типа, таких как формулы прямоугольников, трапеций, Симпсона. Возможно использование как равномерной, так и неравномерной адаптированной сетки.

  3. Прямые методы решения систем линейных алгебраических уравнений. Программа позволяет познакомится с проблемами, возникающими при решении систем линейных алгебраических уравнений методами прогонки. Экспериментируя с программой, можно исследовать влияние погрешностей округления на работу различных вариантов метода прогонки, таких как стандартная, Дорра, немонотонная.

  4. Решение нелинейных алгебраических уравнений итерационными методами. Работая с данным блоком программы можно познакомится с различными итерационными методами решения нелинейных уравнений: методами простой итерации, Ньютона и его модификациями, методом Хелли и его вариантами.

  5. Методы Рунге-Кутты. На серии тестовых задач Коши для нелинейных дифференциальных уравнений иллюстрируется работа методов Рунге-Кутты первого, второго и третьего порядков точности.

  6. Методы решения задач с пограничными слоями. На примере дифференциального уравнения второго порядка с малым параметром при старшей производной можно сравнить работу различных разностных схем для решения линейных дифференциальных задач. Рассматриваются схемы проекционного варианта интегро-интерполяционного метода, а также схемы с центральной и направленной разностями, А.М.Ильина, А.А.Самарского и некоторые другие.

  7. Методы решения задач диффузионного переноса. Для уравнения теплопроводности с разрывной начальной функцией программа позволяет сравнить различные варианты схемы с весами: явной, Кранка-Николсона и схемы с опережением.

  8. Итерационные методы решения двумерных сеточных уравнений. На примере сеточной задачи для двумерного уравнения Пуассона рассматривается двупараметрическое семейство итерационных методов ее решения. Можно познакомиться с методами Якоби, Зейделя, оптимальной верхней релаксации, а также выбрать любой из сходящихся итерационных процессов, варьируя параметры в заданной области.

  1. Самостоятельное выполнение Лабораторной работы по теме, сформулированной преподавателем.

Эта деятельность предполагает разработку и отладку программы, проведение численных экспериментов, написание отчета. Для облегчения работы студент использует методические материалы, которые включают: а) основные алгоритмы, б) тестовые функции (задачи), в) требования к программе, г) задание для работы с программой, д) список дополнительной литературы. В настоящее время имеются методические материалы по следующим темам:

Лаб.раб.№1: Интерполирование квадратичным сплайном.

Лаб.раб.№2: Интерполирование алгебраическими многочленами.

Лаб.раб.№3: Интерполяционный кубический сплайн.

Лаб.раб.№4: Восстановление сеточных функций методом наименьших квадратов (линейная, степенная и показательная регрессии).

Лаб.раб.№5: Метод Гаусса с выбором главного элемента для решения систем линейных алгебраических уравнений.

Лаб.раб.№6: Сглаживание сеточных функций.

Лаб.раб.№7: Сглаживающий кубический сплайн.

Лаб.раб.№8: Методы Рунге-Кутты.

Лаб.раб.№9-10: Методы построения адаптивных сеток.

Лаб.раб.№11: Итерационные методы решения двумерных сеточных уравнений.

Лаб.раб.№12-15: Схема с весами для уравнения теплопроводности.

Лаб.раб.№16-18: Разностные схемы для уравнения конвективного переноса.
